# Cloud Learning Journey

This repository contains the cloud projects and notes I made while learning Linux, AWS, and Terraform.

The main goal of this repo is to track my progress through hands-on practice and small infrastructure projects.

## Projects

### aws-ec2-nginx
This was one of my first AWS projects.

In this project, I launched an Ubuntu EC2 instance, connected to it with SSH, installed Nginx, and tested that the website was reachable through the public IP.

### terraform-nginx
This project is the Terraform version of the EC2 + Nginx setup.

Instead of creating the resources manually in AWS, I used Terraform to create the security group and EC2 instance, and used (user_data) to install Nginx automatically.
